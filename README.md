# StarkBill

A lightweight crypto invoicing system built on Starknet, enabling fast invoice creation and payments using Starkzap.

---

## Overview

StarkBill allows users to:

- Create crypto invoices in seconds
- Generate shareable payment links
- Pay invoices using Starknet wallets via Starkzap
- Track invoice status (pending / paid)

The project is designed for speed, simplicity, and web3-native payments.

---

## Features

- Instant invoice creation
- Unique payment links (`/pay/[id]`)
- Starknet-based payments via Starkzap
- Clean and minimal UI
- Next.js App Router architecture

---

## Tech Stack

- Next.js (App Router)
- TypeScript
- React
- Starknet ecosystem integration
- Starkzap payment layer
